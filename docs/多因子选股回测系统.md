## 策略思想

基本假设：股票收益率可以被多个独立特征维度（因子）解释。在全市场股票池中横向对比（横截面分析）这些指标，筛选综合得分最高的股票构建投资组合，利用概率优势战胜市场基准。

当前版本 V1.0 采用等权重合成法，建立 Baseline，为后续机器学习算法迭代提供参照基准。

---

## 因子库

系统引入四类经典因子，每类一个核心指标：

| 类别 | 指标 | 数据源 | 方向 | 经济学逻辑 |
|------|------|--------|------|------------|
| **价值因子** | PE (市盈率 TTM) | 东财/akshare | 负向 | 寻找低估资产，博取估值修复收益 |
| **价值因子** | PB (市净率) | 东财/akshare | 负向 | 账面价值折价，深度价值信号 |
| **质量因子** | ROE (净资产收益率) | 东财/akshare | 正向 | 筛选高盈利、财务健康的好公司 |
| **动量因子** | 20 日累计收益率 | 新浪 K 线 | 正向 | 趋势跟踪，惯性效应 |
| **规模因子** | ln(总市值) | 东财/akshare | 负向 | 小市值溢价，高弹性 |

## 流水线步骤

```
原始数据 → 硬过滤 → 因子提取 → MAD 去极值 → Z-Score → 方向对齐 → 等权重合成 → Top-N 选股
```

### 步骤 1：数据清洗与硬过滤

在计算因子前剔除无法正常交易的股票：

| 过滤条件 | 规则 | 实现方式 |
|----------|------|----------|
| ST / *ST | 名称含 "ST" | `_hard_filter()` 检查 name |
| 次新股 | K 线不足 120 根（≈上市满 6 个月） | `len(df) < MIN_KLINES` |
| 停牌股 | 最新 K 线距调仓日 > 5 个自然日 | `(rb_date - last_date).days > 5` |

### 步骤 2：横截面预处理

所有预处理在**同一交易日截面内**的股票间独立进行。

**去极值（MAD 法）：**

```
Median = 截面中位数
MAD    = Median(|X - Median|)
Upper  = Median + 3 × 1.4826 × MAD
Lower  = Median - 3 × 1.4826 × MAD
超出 [Lower, Upper] 的值强制拉回边界
```

**标准化（Z-Score）：**

```
Z = (X - Mean) / Std
处理后 μ=0, σ=1
```

### 步骤 3：方向对齐

最终综合得分是"越高越好"，负向因子取反：

```python
PE_score   = -PE_z      # 越低越好 → 取反
PB_score   = -PB_z      # 越低越好 → 取反
MCAP_score = -MCAP_z    # 越小越好 → 取反
ROE_score  = +ROE_z     # 越高越好 → 保持
MOM_score  = +MOM_z     # 越高越好 → 保持
```

### 步骤 4：等权重合成

```
Total_Score = (PE_score + PB_score + ROE_score + MOM_score + MCAP_score) / 有效因子数
```

若某只股票某因子缺失，该因子不得分，分母相应减少——不要求所有股票具备全部因子。

### 步骤 5：Top-N 选股

按 `Total_Score` 降序排列，取前 N 只（默认 30）进入持仓组合。

## 交易回测逻辑

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 调仓频率 | **月频** | 每月第一个交易日开盘调仓 |
| 持仓数量 | **Top-30** | 每期等资金权重，每只 1/30 |
| 初始资金 | **100 万** | `initial_capital = 1_000_000` |
| 买入成本 | **0.11%** | 佣金万1 + 双边滑点各 5bp |
| 卖出成本 | **0.21%** | 佣金万1 + 印花税千1 + 双边滑点各 5bp |

```
买入成本 = 0.0001(佣金) + 0.001(滑点) = 0.0011
卖出成本 = 0.0001(佣金) + 0.001(印花税) + 0.001(滑点) = 0.0021
```

**调仓逻辑：**
1. 每月调仓日运行选股流水线，获得新持仓列表
2. 卖出不在新列表中的股票（扣除卖出成本）
3. 等资金买入新股票（扣除买入成本）
4. 保留已在持仓中的继续持有

**注意：** V1.0 使用固定财务快照（如 2025-12-31 的 PE/PB/ROE/市值），回测期内财务数据不变。这忽略了财务数据的时间变化，引入了一定的前瞻偏差。后续版本每月拉取历史财报快照解决。

## 输出指标

| 指标 | 说明 | 公式 |
|------|------|------|
| **累计收益率** | 期末净值 / 初始资金 - 1 | `final_nav / initial - 1` |
| **年化收益率** | 复利年化 | `(1 + cum_return)^(365/days) - 1` |
| **最大回撤** | 峰值到谷底的最大跌幅 | `max((nav - peak) / peak)` |
| **夏普比率** | 风险调整后收益（无风险利率 0） | `mean(月收益) / std(月收益) × √12` |
| **平均换手率** | 单期平均交易额 / 初始资金 | `total_turnover / (capital × n_periods)` |

## 扩展方向

| 优先级 | 方向 | 说明 |
|--------|------|------|
| 🔴 高 | **多期财务快照** | 每月下载独立快照，消除前瞻偏差 |
| 🔴 高 | **基准对比** | 加入沪深300等基准收益曲线 |
| 🟡 中 | **动态因子权重** | 从等权重升级为 IC 加权 / 信息比率加权 |
| 🟡 中 | **行业中性化** | 因子值减去行业均值，纯化 Alpha |
| 🟡 中 | **周频调仓** | 支持 Weekly 模式（修改 `_monthly_rebalance_dates`） |
| 🟢 低 | **ML 模型** | 用 XGBoost/LightGBM 替代等权重合成 |
| 🟢 低 | **可视化** | 净值曲线、回撤图、因子暴露热力图 |
