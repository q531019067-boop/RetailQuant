## 数据获取与缓存

| 概念 | 定位 | 说明 |
|---|---|---|
| K 线拉取（业务层） | `rquant.business.data.fetch_kline(code, days) → DataFrame` | 从数据源池拉 K 线，失败返回空 DataFrame |
| K 线拉取（池层） | `rquant.data_source.pool.DataSourcePool.fetch_kline(code, days) → list[dict]` | 按源健康度遍历 K 线源 |
| K 线转 DataFrame | `rquant.data_source.pool.DataSourcePool.to_dataframe(code, days)` | 标准列：`date, open, high, low, close, volume` |
| Sina K 线源 | `rquant.data_source.sina.SinaKlineSource` | Sina HTTP + `cache/rquant.db` SQLite K 线缓存 |
| Sina 行情源 | `rquant.data_source.sina.SinaQuoteSource` | 批量实时行情 + QuoteCache |
| 行情缓存 | `rquant.data_source.quote_cache.QuoteCache` | `config.cache.quote_ttl` 秒 TTL + inflight 击穿保护 |
| SQLite 存储 | `rquant.data_source.db` | `klines` / `pool` / `meta` 三张表 |
| Parquet 读取 | `rquant.data_source.parquet_store.read(code)` | 读取 `data/parquet/{code}.parquet` |
| Parquet 写入 | `rquant.data_source.parquet_store.write(code, df, mode)` | replace / append 去重，依赖 `pyarrow` |
| 历史数据拉取 | `scripts/fetch_hist.py` | Sina 全量日 K → Parquet |
| 财务快照下载 | `rquant.data_source.eastmoney.download_snapshot(date)` | akshare 拉全 A PE/PB/ROE/市值 → SQLite |
| 财务快照读取 | `rquant.data_source.eastmoney.get_all_snapshots(date)` | `{code: {pe_ttm, pb, roe, mcap}}` |
| 数据源健康度 | `rquant.data_source.pool.status()` | 返回 K 线源/行情源 healthy 状态 |

|---|---|---|
| 持仓列表 | `rquant.business.portfolio.get_positions()` | 读取 `data/portfolio.json` |

| 卖出持仓 | `rquant.business.portfolio.sell_position(code, shares, price)` | 减仓/清仓，返回 pnl |

| 卖出回款 | `rquant.business.funds.add_on_sell(user_id, proceeds, cost_released)` | 增加 available，释放成本，累计 realized_pnl |
| 充值 | `rquant.business.funds.topup_funds(user_id, amount)` | 增加 total_funds + available_funds |
| 提现 | `rquant.business.funds.withdraw_funds(user_id, amount)` | 最多提到 available 清零 |

## MultiFactor（多策略评分阶段）

`MultiFactor` 是**多策略评分阶段**的注册策略之一：链路中由 `score_stock_strategies()` / `_score_one_for_sim()` 调用 `MultiFactor.score()`，与 `MovingAverageCross` 等并列产出分数与信号。组合模拟阶段只消费这些结果，不重新定义策略。

因子设计、独立验证脚本与报告见下表；端到端链路见 `docs/策略回测链路测试文档.md`。

| 文档 | 代码 | 定位 |
|---|---|---|
| `docs/多因子选股回测系统.md` | `rquant/strategy/factor/multi_factor.py` | 多策略评分阶段：`MultiFactor` 的 8 因子、4 过滤、`score()` 逻辑 |
| `docs/multi_factor_report.md` | `rquant/strategy/factor/multi_factor.py` + `scripts/backtest_multi_factor.py` | 该策略独立验证的回测报告（非链路组合模拟） |
| `docs/策略回测链路测试文档.md` | `rquant/research/workflow.py` + `scripts/select_board_pool.py` 等 | 含多策略评分在内的端到端链路测试 |
| `rquant/strategy/factor/factor_calc.py` | 月频调仓回测（财务因子 TopN） | 2026-06-22 修复：`run_pipeline` 拉 K 线后按 `rebalance_date` 切片防 look-ahead |
