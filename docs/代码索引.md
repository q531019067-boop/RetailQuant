## 数据获取

| 概念 | 定位 | 说明 |
|---|---|---|
| K 线拉取（业务层） | `data.fetch_kline(code, days) → DataFrame` | 从数据源池拉 K 线，返回 OHLCV DataFrame |
| **历史日频 Parquet 存储** | `parquet_store.read(code) → DataFrame` | 读取一只股票的全部历史日频数据（2005 至今） |
| **Parquet 写入** | `parquet_store.write(code, df, mode)` | 写入历史数据，支持 replace / append 去重 |
| **Parquet 文件存在检查** | `parquet_store.exists(code) → bool` | 检查某只股票的 Parquet 缓存是否存在 |
| **Parquet 列表** | `parquet_store.list_codes() → list[str]` | 列出全部已缓存的股票代码 |
| **Parquet 元信息** | `parquet_store.info(code) → dict｜None` | 返回行数、日期范围、文件大小 |
| **历史数据批量拉取** | `scripts/fetch_hist.py` | 从 Sina 拉取全量日 K → Parquet，支持日期过滤 |
| K 线拉取（池层） | `datasources.DataSourcePool.fetch_kline(code, days) → list[dict]` | 按优先级遍历 K 线源 |
| K 线→DataFrame | `datasources.DataSourcePool.to_dataframe(code, days) → DataFrame` | fetch_kline + 转 DataFrame |
| K 线缓存（本地 JSON） | `datasources.SinaKlineSource` | 每个 code 独立 JSON 缓存，5 天过期 |
| 实时行情（单个） | `datasources.DataSourcePool.fetch_quote(code) → dict｜None` | 按优先级取行情 |
| 实时行情（批量） | `datasources.DataSourcePool.fetch_quotes(codes) → dict[str, dict]` | 批量拉行情，走 Sina 批量接口 |
| 行情解析 | `datasources.SinaQuoteSource._parse_line(code, text)` | 解析 Sina JS 变量格式 `hq_str_xxx` |
| 行情批量缓存 | `datasources.SinaQuoteSource.BATCH_TTL = 30s` | 30 秒内同类请求复用 |
| 源码健康度 | `datasources.DataSourcePool.status() → dict` | 返回所有源 healthy 状态 |
| 源失败冷却 | `datasources.UNHEALTHY_COOLDOWN = 60s` | 单源失败后 60 秒内不重试 |

## 数据源协议与池

| 概念 | 定位 | 说明 |
|---|---|---|
| K 线源协议 | `datasources.KlineSource` (Protocol) | 定义 fetch / healthy 接口 |
