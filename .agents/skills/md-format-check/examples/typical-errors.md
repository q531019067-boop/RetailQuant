# 典型错误案例

## 案例 1：表格被代码块包裹

### 错误写法

````
```
| 配置项 | 类型 | 说明 |
|--------|------|------|
| Name | 字符串 | 名称 |
| Value | 整数 | 值 |
```
````

### 正确写法

````
| 配置项 | 类型 | 说明 |
|--------|------|------|
| Name | 字符串 | 名称 |
| Value | 整数 | 值 |
````

## 案例 3：列数不匹配

### 错误写法

````
| 列1 | 列2 | 列3 |
|------|------|------|
| 值1 | 值2 |
````

### 正确写法

````
| 列1 | 列2 | 列3 |
|------|------|------|
| 值1 | 值2 | |
````

---

## 案例 4：分隔符格式错误

### 错误写法

````
| 列1 | 列2 | 列3 |
|------|------|------|
| 值1 | 值2 | 值3 |
````

注意：分隔符行缺少结尾的 `|`。

### 正确写法

````
| 列1 | 列2 | 列3 |
|------|------|------|
| 值1 | 值2 | 值3 |
````
