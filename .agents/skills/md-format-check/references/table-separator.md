# 表格分隔符格式

## 错误现象

表格不被识别，显示为普通文本。

## 错误原因

分隔符行格式不正确。

## 正确格式

分隔符行必须满足：
1. 以 `|` 开头和结尾
2. 每列之间用 `|` 分隔
3. 列内容只能包含 `-`、`:`、空格

## 正确写法

```
|------|------|------|
|:-----|:----:|-----:|
|------|------|------|
```

## 错误写法

```
|------|------|------|    # 正确
|------|------|------      # 错误：结尾缺 |
|------|------|------|    # 正确
|------|------|------x|   # 错误：包含非法字符
```

## 对齐方式

分隔符中的 `:` 表示对齐方式：
- `:-----` 左对齐
- `:----:` 居中对齐
- `-----:` 右对齐
- `------` 默认左对齐

## 检测方法

```bash
# 检测分隔符行格式
grep -n "^|[- ]*|" file.md | while read line; do
  linenum=$(echo "$line" | cut -d: -f1)
  content=$(echo "$line" | cut -d: -f2-)
  if ! echo "$content" | grep -q "^|[-:| ]*|$"; then
    echo "Line $linenum: invalid separator format: $content"
  fi
done
```
