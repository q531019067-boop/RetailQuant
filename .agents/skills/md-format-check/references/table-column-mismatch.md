# 表格列数不匹配

## 错误现象

表格渲染错乱，某些列的内容跑到其他列去了。

## 错误原因

表头行、分隔符行、数据行的列数不一致。

## 检测方法

```bash
awk '/^\|/ {
  n = split($0, a, "|") - 1
  if (prev_n > 0 && n != prev_n) {
    printf "Line %d: column count mismatch (%d vs %d)\n", NR, prev_n, n
  }
  prev_n = n
} !/^\|/ { prev_n = 0 }' file.md
```

## 修复方法

对齐所有行的列数：
- 表头行：`| 列1 | 列2 | 列3 |`
- 分隔符行：`|------|------|------|`
- 数据行：`| 值1 | 值2 | 值3 |`

三行的 `|` 数量必须相同。

## 示例

错误写法：
```
| 列1 | 列2 | 列3 |
|------|------|------|
| 值1 | 值2 |
```

正确写法：
```
| 列1 | 列2 | 列3 |
|------|------|------|
| 值1 | 值2 | |
```
