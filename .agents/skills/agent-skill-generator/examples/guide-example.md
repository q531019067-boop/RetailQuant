# 指导器类型Skill示例

## 功能说明
本示例展示如何生成一个指导器类型的Skill，用于指导Agent调用其他Skill或MCP工具。

## 输入
```
指导agent去调用别的skill或者mcp的指导器
```

## 输出
```
正在生成skill: skill-guide (类型: guide)
创建目录结构...
编写SKILL.md文件...
生成脚本文件...
Skill生成完成！
生成路径: K:\Sword3-products\trunk\tools\SKILLS\skill-guide
```

## 生成的文件结构
```
skill-guide/
├── SKILL.md              # 指导器使用指南
├── scripts/              # [可选] 脚本目录
├── examples/             # [可选] 示例目录
└── references/           # [可选] 参考资料
```

## 使用场景

### 场景1：数据处理流程指导
**用户**：我需要处理玩家数据并检查配置

**指导器**：
1. 首先调用 `player-data-processor` 处理玩家数据
2. 然后调用 `config-checker` 检查配置
3. 最后返回处理结果

### 场景2：工具选择指导
**用户**：如何完成这个任务？

**指导器**：
1. 分析任务需求
2. 推荐合适的工具
3. 指导调用顺序

## 可调用工具

### Skill工具
- `skill-name-1` - 功能说明
- `skill-name-2` - 功能说明

### MCP工具
- `mcp-tool-1` - 功能说明
- `mcp-tool-2` - 功能说明

## 指导原则
1. **工具选择** - 根据任务类型选择最合适的工具
2. **调用顺序** - 按照依赖关系确定调用顺序
3. **错误处理** - 每个步骤都要有错误处理机制

## 核心特点
- 智能工具推荐
- 流程指导
- 调用顺序优化
- 错误处理建议
