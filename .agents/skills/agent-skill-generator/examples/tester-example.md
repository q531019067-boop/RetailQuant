# 测试器类型Skill示例

## 功能说明
本示例展示如何生成一个测试器类型的Skill，用于测试配置文件和脚本的正确性。

## 输入
```
测试配置文件和脚本写法是否正确的测试器
```

## 输出
```
正在生成skill: config-tester (类型: tester)
创建目录结构...
编写SKILL.md文件...
生成脚本文件...
Skill生成完成！
生成路径: K:\Sword3-products\trunk\tools\SKILLS\config-tester
```

## 生成的文件结构
```
config-tester/
├── SKILL.md              # 测试器使用指南
├── scripts/              # [必需] 测试脚本
│   └── run_tests.py      # 测试执行脚本
├── examples/             # [可选] 示例目录
└── references/           # [可选] 参考资料
```

## 使用场景

### 场景1：配置文件语法检查
**用户**：帮我测试这个配置文件

**测试器**：
1. 读取配置文件
2. 检查语法正确性
3. 返回测试结果

### 场景2：数据格式验证
**用户**：验证数据文件格式

**测试器**：
1. 解析数据文件
2. 检查字段类型和范围
3. 生成测试报告

## 测试流程
1. **准备测试环境** - 检查测试环境是否就绪
2. **执行测试** - 运行测试脚本
3. **查看结果** - 分析测试结果

## 核心特点
- 自动检测配置错误
- 验证数据格式
- 生成详细测试报告
- 支持批量测试
