# 转换器类型Skill示例

## 功能说明
本示例展示如何生成一个转换器类型的Skill，用于编码转换、乱码修复、批处理、批量总结和代码文档生成。

## 输入
```
批量转换文件编码，将GBK编码的文件转换为UTF-8编码
```

## 输出
```
正在生成skill: encoding-converter (类型: converter)
创建目录结构...
编写SKILL.md文件...
生成脚本文件...
Skill生成完成！
生成路径: K:\Sword3-products\trunk\tools\SKILLS\encoding-converter
```

## 生成的文件结构
```
encoding-converter/
├── SKILL.md                    # 转换器使用指南
├── scripts/                    # [必需] 转换脚本
│   ├── convert_encoding.py     # 编码转换脚本
│   ├── fix_garbled.py          # 乱码修复脚本
│   └── batch_process.py        # 批处理脚本
├── examples/                   # [可选] 示例目录
└── references/                 # [可选] 参考资料
```

## 支持功能

### 1. 编码转换
批量转换文件编码格式
- 支持GBK、UTF-8、UTF-8 BOM等编码
- 自动检测源编码
- 批量处理目录下的所有文件

### 2. 乱码修复
智能修复文件中的乱码内容
- 分段处理大文件
- 利用AI识别乱码位置
- 精确定位并替换

### 3. 批处理
根据固定工作流批量处理文件
- 支持自定义处理流程
- 批量读取、处理、输出
- 支持过滤和筛选

### 4. 批量总结
根据规则批量扫描文件并汇总
- 提取关键信息
- 生成汇总报告
- 支持多种输出格式

### 5. 代码文档生成
为代码自动生成文档
- 解析代码结构
- 提取函数说明
- 按照指定格式输出

## 使用场景

### 场景1：编码转换
**用户**：将./data目录下的所有GBK编码文件转换为UTF-8编码

**转换器**：
1. 扫描目录中的所有文件
2. 检测文件编码
3. 转换为目标编码
4. 输出转换结果

### 场景2：乱码修复
**用户**：修复文件中的乱码

**转换器**：
1. 分段读取文件内容
2. 利用AI识别乱码位置
3. 定位并替换乱码
4. 继续处理下一段

### 场景3：批处理
**用户**：按照固定工作流批量处理文件

**转换器**：
1. 加载工作流配置
2. 批量读取文件
3. 按照流程处理
4. 输出处理结果

### 场景4：代码文档生成
**用户**：为所有Lua导出函数生成文档

**转换器**：
1. 扫描代码目录
2. 解析Lua函数
3. 提取函数说明
4. 按照指定格式生成文档

## 核心特点
- 支持多种转换功能
- 批量处理能力
- 智能乱码修复
- 代码文档自动生成
- 工作流支持
