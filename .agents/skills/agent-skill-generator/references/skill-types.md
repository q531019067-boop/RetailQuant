# Skill类型映射表

## 说明
本文件包含了agent-skill-generator支持的所有Skill类型及其对应的关键词。

## Skill类型定义

### 1. 转换器类型 (converter)
**功能**：用于编码转换、乱码修复、批处理、批量总结、代码文档生成
**关键词**：转换、编码、乱码、批处理、批量处理、批量总结、文档生成、生成器
**生成内容**：
- SKILL.md - 转换器使用指南
- scripts/convert_encoding.py - 编码转换脚本
- scripts/fix_garbled.py - 乱码修复脚本
- scripts/batch_process.py - 批处理脚本

### 2. 工具类型 (tool)
**功能**：用于处理数据的工具，如编辑、修改、导出数据等
**关键词**：处理、管理、编辑、创建、修改、读取、批量、导出
**生成内容**：
- SKILL.md - 工具使用指南
- scripts/main.py - 示例脚本

### 3. 知识库类型 (knowledge_base)
**功能**：用于回答配置相关问题，提供知识和规则说明
**关键词**：知识库、文档、回答、查询、解释、说明
**生成内容**：
- SKILL.md - 知识库使用指南
- references/config-guide.md - 配置指南

### 4. 测试器类型 (tester)
**功能**：用于测试配置文件和脚本的正确性
**关键词**：测试、验证、检测、试运行
**生成内容**：
- SKILL.md - 测试器使用指南
- scripts/run_tests.py - 测试脚本

### 5. 检查器类型 (checker)
**功能**：用于检查配置文件的正确性和完整性
**关键词**：检查、校验、审查、核对、排查
**生成内容**：
- SKILL.md - 检查器使用指南
- scripts/check.py - 检查脚本

### 6. 指导器类型 (guide)
**功能**：用于指导Agent调用其他Skill或MCP工具
**关键词**：指导、指引、调用、使用、协助
**生成内容**：
- SKILL.md - 指导器使用指南（包含工具调用指南）

### 7. 流程专家类型 (planner)
**功能**：用于指导生成ReAct Agent的工作流程计划
**关键词**：计划、规划、流程、步骤、方案、ReAct
**生成内容**：
- SKILL.md - 流程专家使用指南（包含ReAct流程模板）

### 8. 复合型 (hybrid)
**功能**：同时具备多种角色，如知识库+检查器+查询器+指导器
**关键词**：专家、复合、多功能、综合
**生成内容**：
- SKILL.md - 复合型专家使用指南
- references/ - 相关参考文档

## 类型识别规则
1. 优先匹配第一个匹配的关键词
2. 如果没有匹配的关键词，默认使用tool类型
3. 支持多种关键词组合，但只返回第一个匹配的类型
