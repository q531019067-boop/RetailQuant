# LaTeX 数学公式 Markdown 规则参考

## 1. 行内公式（Inline Math）

### 语法
- 使用单个 `$` 包裹：`$...$`
- 公式嵌入在文本行中，与周围文字同行显示
- 适用于短小的数学表达式

### 示例
```markdown
变量 $x$ 的取值范围是 $[0, 1]$。
当 $n \to \infty$ 时，序列收敛。
```

### 空格规则
- `$` 与公式内容之间 **不能有空格**
- 错误：`$ x$`、`$x $`、`$ x $`
- 正确：`$x$`
- `$` 前后与文本之间可以有空格（推荐），也可以没有

### 适用场景
- 单个变量、简短表达式
- 不超过一行高度的公式（避免 `\frac` 导致行高膨胀）

## 2. 块级公式（Block Math）

### 语法
- 使用双 `$$` 包裹：`$$...$$`
- **必须独占一行**，前后不能有其他文本
- 公式前后建议各有一个空行

### 示例
```markdown
总和公式如下：

$$
S = \sum_{i=1}^{n} a_i
$$

由上式可知...
```

### 常见错误
```markdown
<!-- 错误：$$ 前面有文本 -->
公式的写法是 $$x = y$$

<!-- 错误：$$ 后面有文本 -->
$$x = y$$ 这是公式

<!-- 正确 -->
$$
x = y
$$
```

## 3. 支持的 LaTeX 命令

### 基础运算
| 命令 | 说明 | 示例 |
|------|------|------|
| `\frac{a}{b}` | 分数 | `$\frac{1}{2}$` |
| `\sqrt{x}` | 平方根 | `$\sqrt{2}$` |
| `\sqrt[n]{x}` | n 次根 | `$\sqrt[3]{8}$` |
| `^{}` | 上标 | `$x^{2}$` |
| `_{} ` | 下标 | `$a_{n}$` |

### 求和与积分
| 命令 | 说明 | 示例 |
|------|------|------|
| `\sum` | 求和符号（注意：不是 `\Sigma`） | `$\sum_{i=1}^{n} a_i$` |
| `\prod` | 连乘符号 | `$\prod_{i=1}^{n} a_i$` |
| `\int` | 积分符号 | `$\int_{0}^{1} f(x) dx$` |
| `\lim` | 极限 | `$\lim_{x \to 0} \frac{\sin x}{x}$` |

### 希腊字母
| 命令 | 字母 | 命令 | 字母 |
|------|------|------|------|
| `\alpha` | α | `\Delta` | Δ |
| `\beta` | β | `\Sigma` | Σ（大写，非求和） |
| `\gamma` | γ | `\Omega` | Ω |
| `\theta` | θ | `\lambda` | λ |

### 特殊符号
| 命令 | 说明 | 示例 |
|------|------|------|
| `\ldots` | 省略号 | `$a_1, a_2, \ldots, a_n$` |
| `\cdot` | 点乘 | `$a \cdot b$` |
| `\vert` | 竖线（推荐） | `$\{x \vert x > 0\}$` |
| `\mid` | 竖线（**不要在表格中使用**） | `$\{x \mid x > 0\}$` |
| `\leq` | 小于等于 | `$x \leq 10$` |
| `\geq` | 大于等于 | `$x \geq 0$` |
| `\neq` | 不等于 | `$a \neq b$` |
| `\in` | 属于 | `$x \in \mathbb{R}$` |

### 文字插入
| 命令 | 说明 | 用法 |
|------|------|------|
| `\text{...}` | 文本模式（推荐，支持中文） | `$x = \text{最大值}$` |
| `\mathrm{...}` | 罗马体（**不支持中文**） | `$\mathrm{const}$` |
| `\textrm{...}` | 罗马体（**不支持中文**） | 同上 |

## 4. 不支持 / 有问题的写法

### `{,}` 千分位分隔符 — 错误
```markdown
<!-- 错误：某些渲染器无法解析 -->
$1{,}000{,}000$

<!-- 正确：直接写数字 -->
$1000000$

<!-- 或者用文字 -->
100万
```

### `\Sigma` vs `\sum` — 混淆
```markdown
<!-- 错误：\Sigma 是大写希腊字母 -->
$\Sigma_{i=1}^{n} a_i$

<!-- 正确：\sum 是求和符号 -->
$\sum_{i=1}^{n} a_i$
```
- `\Sigma` 显示为大写的 Σ，没有上下限位置
- `\sum` 显示为求和符号 ∑，上下限自动放在上方和下方

### `\mathrm{}` / `\textrm{}` 包含中文 — 乱码
```markdown
<!-- 错误：中文会变成乱码 -->
$\mathrm{最大值}$

<!-- 正确：使用 \text{} -->
$\text{最大值}$
```

## 5. 表格中的公式约束

### 绝对禁止
- **不要使用 `|`**（竖线），它会被解析为表格列分隔符
- **不要使用 `\mid`**，在某些渲染器中同样有冲突

### 推荐做法
- 使用 `\vert` 替代 `|` 和 `\mid`
- 尽量简化公式，避免复杂的行内数学
- 如果公式复杂，放到表格外部用块级公式展示

### 示例
```markdown
<!-- 错误 -->
| 条件 | 公式 |
|------|------|
| 正数 | $x \mid x > 0$ |

<!-- 正确 -->
| 条件 | 公式 |
|------|------|
| 正数 | $x \vert x > 0$ |
```

## 6. 数字格式化

- 不要在公式中使用 `{,}` 作千分位
- 直接写数字：`$1000000$` 而非 `$1{,}000{,}000$`
- 大数字建议用科学计数法：`$10^6$`
- 或者将数字放到公式外用文字表示

## 7. 中文在公式中的处理

### 首选方案：中文放在公式外
```markdown
<!-- 推荐 -->
公式 $E = mc^2$ 表示能量与质量的关系。
其中 $E$ 代表能量，$m$ 代表质量。
```

### 次选方案：使用 `\text{}`
```markdown
<!-- 可接受 -->
$P(\text{成功}) = \frac{1}{2}$
```

### 避免的方案
```markdown
<!-- 不要使用 \mathrm 或 \textrm 包含中文 -->
$\mathrm{成功概率}$
$\textrm{失败}$
```

### 注意事项
- `\text{中文}` 在 MathJax 和 KaTeX 中基本可用，但可能出现对齐问题
- 复杂的行内公式（含 `\frac` + 中文 `\text{}`）会导致行高不一致，建议转为块级公式
