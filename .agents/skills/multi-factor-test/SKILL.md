# 多因子策略回测与打分工作流 (Multi-Factor Test Workflow)

本 Skill 规范了 RetailQuant 系统中进行多因子策略分析、历史数据拉取、批量因子计算打分、横截面排序、调仓回测以及最终报告产出落盘的标准工作流。它指导 AI 代理如何协同调用底层 SQLite 缓存库、Parquet 历史列存以及策略注册器，确保整个分析过程具备严密的数据闭环和可复现性。
