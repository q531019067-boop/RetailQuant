# 解方程方法指南

> 从小学一元一次到游戏数值平衡——解方程是数学最古老也最核心的应用。

## 方法分类

### 1. 符号求解（精确解）

**适用**：多项式方程、可因式分解的方程、线性方程组

**工具**：`calc_sym.py --op solve`（单个方程或方程组）

**原理**：利用 sympy 的代数求解引擎，通过因式分解、配方法、求根公式等方法给出精确解。

**示例**：

- 二次方程：$x^2 - 5x + 6 = 0 \Rightarrow x = 2$ 或 $x = 3$
- 方程组：$x + y = 10, \; x - y = 2 \Rightarrow x = 6, y = 4$

**局限性**：五次及以上多项式一般无根式解（阿贝尔-鲁菲尼定理）。高次方程推荐数值方法。

### 2. 线性方程组（矩阵法）

**适用**：$A\mathbf{x} = \mathbf{b}$ 形式的线性方程组

**工具**：`calc_sym.py --op linsolve`

**方法**：

- **LU 分解**：$A = LU$，先解 $L\mathbf{y} = \mathbf{b}$ 再解 $U\mathbf{x} = \mathbf{y}$，$O(n^3)$
- **Gauss-Jordan 消元**：增广矩阵行变换到最简形
- **Cramer 法则**：$x_i = \frac{\det(A_i)}{\det(A)}$（仅适用于小规模，$O(n!)$）

**示例**：
$$\begin{cases} 3x + 2y = 12 \\ x - y = -1 \end{cases} \Rightarrow x = 2, y = 3$$

**游戏应用**：属性分配优化（DPS 最大化 = 解一组偏导数方程）。

### 3. 数值求根（近似解）

**适用**：无法解析求解的方程（超越方程、高次方程）

**工具**：`series_tools.py --op newton` / `--op bisect`

| 方法 | 收敛速度 | 要求 |
|------|---------|------|
| 牛顿法 | 二次收敛（极快） | 需要导数，需要好初值 |
| 二分法 | 线性收敛（稳） | 需要区间两端异号 |

**牛顿迭代公式**：


$$
x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}
$$

**二分法**：每次将区间缩小一半，保证收敛但速度较慢。$|b_n - a_n| = \frac{b_0 - a_0}{2^n}$。

### 4. 常微分方程（ODE）

**适用**：包含导数的方程——描述变化率的关系

**工具**：`calc_sym.py --op dsolve`（符号）/ `--op ode_numeric`（数值 RK45）

**分类**：

| 类型 | 形式 | 解法 |
|------|------|------|
| 可分离变量 | $\frac{dy}{dx} = f(x)g(y)$ | 分离积分 |
| 一阶线性 | $\frac{dy}{dx} + P(x)y = Q(x)$ | 积分因子法 |
| 二阶常系数 | $ay'' + by' + cy = 0$ | 特征方程法 |

**数值解示例**（指数增长）：$\frac{dy}{dx} = y, \; y(0) = 1 \Rightarrow$ 用 RK45 求 $y(1) \approx e \approx 2.718$

### 5. 不等式求解

**适用**：寻找变量满足不等关系的区间

**工具**：`calc_sym.py --op inequality`

**示例**：$x^2 - 5x + 6 < 0 \Rightarrow x \in (2, 3)$（抛物线在 x 轴下方的部分）

### 6. 非线性方程组（数值）

**适用**：多个联立非线性方程

**工具**：`calc_sym.py --op fsolve`

**示例**：
$$\begin{cases} x^2 + y^2 = 25 \\ x \cdot y = 12 \end{cases} \Rightarrow x = 3, y = 4$$

## 解析求解策略速查

| 策略 | 适用场景 | 工具 |
|------|---------|------|
| 代入消元 | 一个变量可表示 | `calc_sym solve` |
| 因式分解 | 多项式可分解 | `calc_sym simplify + solve` |
| 配方法 | 二次方程 | `calc_sym solve` |
| 求根公式 | 二次/三次/四次 | `calc_sym solve` |
| 数值迭代 | 超越方程 | `series_tools newton` |
| 图形法 | 找近似解范围 | `plot_tools curve` |
| 矩阵法 | 线性方程组 | `calc_sym linsolve` |

## 游戏中的应用

- **伤害公式反推**：已知期望伤害 $= 1000$，暴击率 $= 0.3$，暴击倍率 $= 2.0$，反推基础攻击力 $x$：


$$
x \cdot (1 + 0.3 \times 2.0) = 1000 \Rightarrow x = 625
$$

- **掉落率校准**：已知保底机制 + 观测到的实际掉落率 → 反推基础概率

- **经济均衡**：供给 $=$ 需求 → 解出均衡价格

- **碰撞时间**：两个运动物体何时碰撞 → 解二次方程 $s_1(t) = s_2(t)$

- **寻路代价**：A* 启发式函数的参数调优 → 解优化方程

- **物理模拟**：弹簧/阻尼系统 → 解二阶 ODE
