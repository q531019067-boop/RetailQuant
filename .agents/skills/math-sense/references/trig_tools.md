# 三角函数全系工具参考

## 概述

`trig_tools.py` 提供三角函数和三角形几何的完整计算工具链，覆盖从基础恒等式到高级分析的 13 个操作。

| 操作 | 功能 | 输入 | 输出 |
|------|------|------|------|
| `verify_identity` | 三角恒等式验证 | lhs表达式 + rhs表达式 | 符号匹配 + 数值误差 |
| `euler` | 欧拉公式验证 | 可选表达式 | 数值验证 + 泰勒展开 + 几何解释 |

## 海伦公式

```
已知三角形三边长 a, b, c：
半周长 s = (a + b + c) / 2
面积 S = √(s(s-a)(s-b)(s-c))

这是已知三边求面积的唯一通用公式，约公元60年由海伦在《度量论》中给出。
```

## 正弦定理

```
a / sin(A) = b / sin(B) = c / sin(C) = 2R

其中 R 为外接圆半径。正弦定理建立了三角形的边与对角的统一比例关系。
当已知两角一边 (ASA) 或两边一对角 (SSA) 时使用。
```

## 余弦定理

```
c² = a² + b² - 2ab·cos(C)

当 C=90° 时，cos(90°)=0，公式退化为 c² = a² + b²（勾股定理）。
当已知两边及其夹角 (SAS) 或已知三边 (SSS) 时使用。
```

## 三角形解算模式

| 模式 | 已知条件 | 解算方法 |
|------|----------|----------|
| SSS | 三边长 a, b, c | 余弦定理求三角 → 海伦公式求面积 |
| SAS | 两边 a, b 及夹角 C | 余弦定理求 c → 正弦定理求 A, B |
| ASA | 三角及夹边 | 先求第三角 → 正弦定理求 a, c |
| SSA* | 两边 a, b 及一角 A | 正弦定理 → 可能有两解或无解 |

*SSA 是"歧义情况"——给定两边和非夹角时可能有两个不同的三角形。

## 游戏应用

- **视野判定**：已知两个角色的位置和朝向 → 用余弦定理计算相对角度 → 判断是否在视野锥内
- **AOI 范围**：已知 NPC 位置和目标位置 → 用海伦公式计算三角形面积 → 判断是否在区域范围内
- **弹道计算**：已知发射点和目标的坐标差 → 用余切线定理求发射角度 → 计算是否存在弹道解
- **三角形网格**：3D 模型的三角面片 → 用海伦公式求每个三角面面积 → 用于碰撞检测和光照计算

## 验证方法

1. **符号化简**：用 sympy 化简恒等式差值为 0（严格证明）
2. **数值采样**：在 50 个随机点比较两边值（概率验证）
3. **极端值测试**：代入 0, π/2, π 等特殊角验证（边界验证）
4. **交叉验证**：同一三角形用 SSS → 余弦定理 → 结果与正弦定理一致
