# 微积分知识库（同济教材方法体系）

> 基于同济大学《高等数学》第七版的证明方法论和经典例子。
> 知识库主体在 `formula_desc.py` 微积分 + 方法论条目中，本文档补充教材特有的证明思路和易错点。

## 1. 极限的 ε-δ 定义（精确化语言）

**核心思想**：用不等式精确描述"无限趋近"。

```
lim(x→x₀) f(x) = A
⟺ ∀ε>0, ∃δ>0, 当 0<|x-x₀|<δ 时, |f(x)-A|<ε
```

**方法论**：ε-δ 是分析学的"通用语言"——把模糊的"趋近"转化为可计算的"误差控制"。给定一个容许误差 ε，总能找到一个足够小的邻域 δ，使得函数值与极限的偏差不超过 ε。

**同济经典例子**：证明 lim(x→x₀) c = c（常数的极限是它自己）。取任意 δ>0（如 δ=1），当 0<|x-x₀|<δ 时，|f(x)-A|=|c-c|=0<ε 恒成立。

**易错点**：δ 的选择通常依赖于 ε——先给 ε，后找 δ。常见的找 δ 技巧：放大 |f(x)-A| 使其 ≤ K|x-x₀|，取 δ=ε/K。

## 2. 夹逼准则（三明治定理）

**核心思想**：如果 g(x) ≤ f(x) ≤ h(x)，且 g 和 h 极限相同为 A，则 f 的极限也是 A。

**同济经典例子** — 重要极限 lim(x→0) sin(x)/x = 1：
- 几何构造：单位圆中，sin x ≤ x ≤ tan x
- 变形：cos x ≤ sin x/x ≤ 1
- 夹逼：当 x→0 时 cos x→1，所以 sin x/x→1

**方法论推广**：当直接求极限困难时，寻找上下界函数。关键在于"放大"和"缩小"要适度——放太大或缩太小都会失去夹逼效果。

---

## 3. 单调有界准则

**核心思想**：单调递增+有上界 ⇒ 极限存在；单调递减+有下界 ⇒ 极限存在。

**同济经典例子** — 重要极限 lim(n→∞) (1+1/n)^n = e：
- 证明数列 x_n = (1+1/n)^n 单调递增且有上界
- 二项式展开 + 放缩法证明上界 < 3
- 由单调有界准则，极限存在，定义为 e ≈ 2.71828...

**方法论**：这是处理递推定义数列的基本工具——先证单调再证有界，极限自然存在。常用于数值分析中的迭代收敛性证明。

## 4. 中值定理族（罗尔→拉格朗日→柯西递进关系）

同济教材以递进方式组织三个中值定理：

| 定理 | 条件 | 结论 | 几何意义 |
|------|------|------|----------|
| 罗尔 | f(a)=f(b) | ∃ξ: f'(ξ)=0 | 水平切线 |
| 拉格朗日 | 无额外条件 | ∃ξ: f'(ξ)=[f(b)-f(a)]/(b-a) | 切线∥割线 |
| 柯西 | 参数形式 | ∃ξ: f'(ξ)/g'(ξ)=[f(b)-f(a)]/[g(b)-g(a)] | 参数曲线切线∥割线 |

**递进关系**：罗尔是拉格朗日的特例（f(a)=f(b)），拉格朗日是柯西的特例（g(x)=x）。证明也递进——用罗尔证拉格朗日（构造辅助函数），用拉格朗日证柯西。

**方法论**：辅助函数法是微分中值定理证明的核心技巧——将待证等式移项，构造一个满足罗尔定理条件的函数。

## 5. 洛必达法则的使用条件（同济强调的易错点）

**条件检查清单**：
1. ✅ 必须是 0/0 或 ∞/∞ 型不定式
2. ✅ f 和 g 在去心邻域内可导
3. ✅ g'(x) ≠ 0
4. ✅ lim f'/g' 存在（或为无穷大）

**常见错误**：
- ❌ 非不定式直接用洛必达（如 lim(x→0) (x+1)/2）
- ❌ 求导后极限不存在就认为原极限不存在（洛必达只是充分条件）
- ❌ 循环使用洛必达跳不出来（如 lim(x→∞) (x+sin x)/(x+cos x)——每次求导不定式不消失，改用夹逼准则）
- ❌ 未验证可导性就用

**替代法**：当洛必达失效时，考虑泰勒展开、等价无穷小替换、夹逼准则。

## 6. 泰勒公式的拉格朗日余项

**公式**：f(x) = P_n(x) + R_n(x)，其中 R_n(x) = f^(n+1)(ξ)/(n+1)! · (x-a)^(n+1)

**方法论**：余项回答了"逼近有多精确"——它给出了误差的确定上界。在数值计算中，余项决定截断误差。

**同济经典例子**：用泰勒公式近似计算 √e：
- f(x)=e^x，在 x=0 处展开到 n 阶
- 余项 R_n(1/2) = e^ξ/(n+1)! · (1/2)^(n+1)
- 由此估算所需阶数以达到指定精度

**与其对比**：佩亚诺余项只告诉误差是 (x-a)^n 的高阶无穷小（定性），拉格朗日余项给出具体上界（定量）。

## 7. 定积分的换元法与对称性

**定积分换元公式**：∫_a^b f(x)dx = ∫_α^β f[φ(t)]·φ'(t)dt

**三个同步**：换元时三样东西同步变——被积函数、积分变量、积分上下限。

**对称区间简化**（同济经典技巧）：
- f 为偶函数：∫_(-a)^a f(x)dx = 2∫_0^a f(x)dx
- f 为奇函数：∫_(-a)^a f(x)dx = 0

**易错点**：换元后的新变量 t 必须单值、单调、有连续导数。如果 φ'(t) 在某处为零，换元公式失效，需要分段处理。

## 8. 反常积分的审敛法

同济教材对反常积分的审敛（判断收敛/发散）提供了系统方法：

| 方法 | 适用 | 思路 |
|------|------|------|
| 比较审敛法 | 正函数 | 找一个已知收敛/发散的参照函数 |
| 极限审敛法 | 正函数 | 比较被积函数在奇点附近的行为 |
| 绝对收敛 | 任意函数 | |f| 收敛 ⇒ f 收敛 |

**经典对比**：∫_1^∞ 1/x^p dx —— p>1 收敛，p≤1 发散。这是比较审敛法的"标尺"。
