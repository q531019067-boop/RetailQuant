# 补充领域：数论、组合、微分方程

> 这些领域在游戏引擎中有实际应用，但知识库尚未覆盖。通过 `formula_desc search` 仍可查询。
> 以下列出核心概念和一键查询命令。

## 数论

| 概念 | 一句话 | 游戏应用 |
|------|--------|---------|
| 素数 | 只能被1和自身整除的正整数 | 随机种子、哈希散列 |
| 同余 | a≡b (mod m)，在模m下相等 | 循环计时、环形缓冲区索引 |
| 最大公约数 GCD | gcd(a,b)=max{d: d|a且d|b} | 宽高比简化、网格对齐 |
| 模逆元 | a·a⁻¹≡1 (mod m) | 伪随机数生成(LCG)、加密 |
| 中国剩余定理 | 联立同余方程组的解法 | 大型数值的分段存储 |
| 欧拉函数 φ(n) | 小于n且与n互素的数的个数 | RSA加密基础 |

工具：`eval_expr` 可做模运算（`%`）、`calc_sym` 可做因式分解。

## 组合数学

| 概念 | 一句话 | 游戏应用 |
|------|--------|---------|
| 排列 P(n,k) | n个元素取k个有序排列数 | 技能连招顺序数 |
| 组合 C(n,k) | n个元素取k个无序组合数 | 掉落'N中取K'概率 |
| 容斥原理 | |A∪B|=|A|+|B|-|A∩B| | 多重条件触发概率 |
| 鸽巢原理 | n+1个物品放入n个盒子，必有一盒≥2个 | 保底机制最小保证次数 |
| 二项式定理 | (a+b)^n = ΣC(n,k)a^k b^(n-k) | 概率展开 |
| 卡特兰数 | 合法括号序列数/二叉树数 | 技能树分支结构计数 |

工具：`eval_expr --expr "factorial(n)/(factorial(k)*factorial(n-k))"` 计算组合数。

## 微分方程

| 概念 | 一句话 | 游戏应用 |
|------|--------|---------|
| 一阶ODE | dy/dx = f(x,y)，分离变量或积分因子求解 | 指数衰减、冷却恢复 |
| 二阶ODE | d²y/dx² + a dy/dx + by = 0 | 弹簧阻尼系统、相机弹性跟随 |
| 欧拉方法 | y_{n+1} = y_n + h·f(x_n,y_n) | 最简单数值解ODE |
| RK4方法 | 四阶龙格-库塔，精度O(h⁴) | 物理模拟（位置/速度更新） |
| 特征方程法 | 二阶常系数ODE的特征根λ求解 | 分析系统稳定性 |
| 拉普拉斯变换 | 将ODE转为代数方程 | 控制系统分析 |

工具：`calc_sym --op "integrate"` 可解一阶ODE；`series_tools` 的牛顿法可用于隐式ODE。

## 数值线性代数

| 概念 | 一句话 | 游戏应用 |
|------|--------|---------|
| 高斯-赛德尔迭代 | 迭代求解大型稀疏线性方程组 | 物理模拟的约束求解 |
| 共轭梯度法 | 对称正定矩阵的最速下降优化 | 大规模稀疏系统 |
| 幂迭代 | 求矩阵最大特征值 | PageRank计算 |
| QR迭代 | 求所有特征值 | 主成分分析 |

工具：`vec_ops --op "mat_eigen"` 可计算特征值。

## 控制论与系统理论

| 概念 | 一句话 | 游戏/工程应用 |
|------|--------|-------------|
| PID控制 | P(比例)+I(积分)+D(微分)——一条公式稳住一切 | 无人机飞控、自动驾驶、相机跟随 |
| 卡尔曼滤波 | 从噪声测量中递归估计系统状态 | GPS定位、IMU姿态解算、目标追踪 |
| 状态空间模型 | x'=Ax+Bu, y=Cx——用一阶微分方程组描述动态系统 | 物理模拟、AI行为状态机 |
| 传递函数 | 频域下系统输入输出的比值 | 滤波器设计、音频处理 |
| 拉普拉斯变换 | 将微分方程转为代数方程 | 分析系统稳定性（极点位置） |
| 李雅普诺夫稳定性 | 找"能量函数"证明其递减→系统稳定 | AI行为收敛性、经济系统均衡 |
| 最优控制(LQR) | 在约束下最小化二次代价函数 | 路径规划、能耗最优轨迹 |
| 反馈回路 | 输出反馈到输入，形成闭环 | 所有控制系统的基础结构 |

> 控制论的浪漫：比例是现在，积分是过去，微分是未来——一条不到10行的PID公式让你的无人机在狂风中纹丝不动。
> 详细故事见 `references/stories/pid-control.md` 和 `references/stories/kalman-filter.md`
