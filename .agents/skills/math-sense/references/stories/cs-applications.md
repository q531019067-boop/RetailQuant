# 数学在计算机科学中的经典应用

> 8 个案例，每个都展示"纯数学公式如何变成改变世界的算法"。

---

## 1. SVD → 推荐系统（Netflix / TikTok / YouTube）

**核心公式**：给定用户-物品评分矩阵 $R_{m \times n}$，奇异值分解：


$$
R = U \Sigma V^T
$$

其中 $U$ 是 $m \times r$ 用户偏好矩阵，$\Sigma$ 是 $r \times r$ 奇异值对角阵，$V^T$ 是 $r \times n$ 物品特征矩阵。

**降维预测**：保留前 $k$ 个最大奇异值（截断 SVD），用 $\hat{R} = U_k \sum_k V_k^T$ 近似原矩阵。未被用户评分的物品，其预测评分 = 对应位置在 $\hat{R}$ 中的值。

**几何直觉**：$U$ 的每一行是用户在"隐藏偏好空间"中的坐标，$V$ 的每一行是物品在同一个隐藏空间中的坐标。用户对物品的评分 = 两个向量的内积。SVD 就是"找到最佳的低维隐藏空间"。

**2009 Netflix Prize**：参赛队 BellKor 用 SVD 改进版将预测误差 RMSE 从 0.95 降到 0.86——百万美元奖金。

## 2. 维特比算法 → 拼音输入法 / 语音识别

**问题**：已知观测序列 $O = (o_1, \ldots, o_T)$ 和隐马尔可夫模型参数，求最可能的隐藏状态序列。

**动态规划递推**：


$$
\delta_t(j) = \max_{i} \left[ \delta_{t-1}(i) \cdot a_{ij} \right] \cdot b_j(o_t)
$$

其中 $\delta_t(j)$ = 到时刻 $t$ 以状态 $j$ 结尾的最优路径概率，$a_{ij}$ = 转移概率，$b_j(o_t)$ = 发射概率。

**回溯路径**：$\psi_t(j) = \arg\max_i [\delta_{t-1}(i) \cdot a_{ij}]$ 记录最优前驱。

**输入法实例**：观测 $O$ = 拼音序列 "wo shi zhong guo ren"，隐藏状态 = 汉字序列 "我是中国人"。维特比用汉字之间的转移概率（语言模型）和拼音到汉字的发射概率（声学模型）找最可能汉字串。复杂度 $O(T \cdot N^2)$，N 是状态数。

## 3. FFT → JPEG 图像压缩

**核心**：每张 $8 \times 8$ 像素块做二维 DCT（离散余弦变换，FFT 的实数变体）：


$$
F(u,v) = \frac{1}{4}C(u)C(v)\sum_{x=0}^{7}\sum_{y=0}^{7} f(x,y) \cos\frac{(2x+1)u\pi}{16} \cos\frac{(2y+1)v\pi}{16}
$$

**压缩原理**：人眼对高频分量（细节）不敏感。量化步长在高频处更大——$F(7,7)$ 通常量化为 0。熵编码（Huffman 编码）对连续的 0 用游程编码大幅压缩。典型 JPEG 可压缩到原图的 $1/10 \sim 1/20$。

## 4. 布隆过滤器 → Redis 缓存穿透 / Chrome 恶意 URL

**数据结构**：$m$ 比特的数组，$k$ 个独立哈希函数。插入元素 $x$：


$$
\text{对于 } i = 1,\ldots,k:\quad \text{位}[h_i(x) \bmod m] = 1
$$

查询 $x$ 是否存在：若所有 $k$ 个位置都是 1 → "可能存在"；任一位置为 0 → "一定不存在"。

**误判率**（插入 $n$ 个元素后）：


$$
P(\text{false positive}) \approx \left(1 - e^{-kn/m}\right)^k
$$

最优哈希函数数 $k = \frac{m}{n}\ln 2$ 时，误判率最小 = $(0.6185)^{m/n}$。

**为什么有用**：不会漏报（"不在"是真的不在）→ 适合做"快速排除"。Redis 用布隆过滤器防止缓存穿透（查一个不存在的 key 直接返回，不查 DB）。Chrome 用布隆过滤器快速判断 URL 是否在恶意网站列表中。

## 5. 梯度下降 → 所有深度学习训练

**优化问题**：最小化损失函数 $J(\theta)$，其中 $\theta$ 是模型参数向量。

**更新规则**：


$$
\theta_{t+1} = \theta_t - \eta \cdot \nabla J(\theta_t)
$$

其中 $\eta$ 是学习率，$\nabla J$ 是损失函数的梯度（对所有参数的偏导数向量）。

**为什么有效**：梯度指向函数值**增长最快**的方向 → 沿负梯度方向走就是**下降最快**的方向。反复执行直到梯度接近零（局部极小值，或鞍点）。

**反向传播**：链式法则计算每层梯度 $\frac{\partial J}{\partial w_{ij}^{(l)}} = \delta_j^{(l)} \cdot a_i^{(l-1)}$，从输出层逐层向输入层传播误差 $\delta$。

AlphaGo 的神经网络有数百万参数——不可能解析求解最优点。梯度下降（配上动量、Adam 等优化器）是在这个百万维空间中"滑向最优"的标准方法。

## 6. 蒙特卡洛树搜索 → AlphaGo 决策

**UCB 公式**（树节点选择策略）：


$$
UCB = \frac{w_i}{n_i} + c \cdot \sqrt{\frac{\ln N}{n_i}}
$$

其中 $w_i$ = 节点 $i$ 的获胜次数，$n_i$ = 节点 $i$ 的访问次数，$N$ = 父节点总访问次数，$c \approx \sqrt{2}$。

**两项的含义**：
- $\frac{w_i}{n_i}$ = **利用**（exploitation）— 选当前看起来最好的走法
- $\sqrt{\frac{\ln N}{n_i}}$ = **探索**（exploration）— 对访问次数少的走法给予"探索奖金"

**AlphaGo 流程**：① 选择（用 UCB 选最有潜力的落子）② 扩展（展开新节点）③ 模拟（用快速策略网络模拟棋局到终局）④ 回传（将胜负结果沿路径回传更新统计）。四步循环百万次后，访问次数最多的落子 = 最优着法。

## 7. RSA 加密 → HTTPS / 在线支付

**密钥生成**：
1. 选两个大素数 $p, q$，计算 $n = p \cdot q$，$\phi(n) = (p-1)(q-1)$
2. 选 $e$ 满足 $\gcd(e, \phi(n)) = 1$（公钥指数）
3. 计算 $d = e^{-1} \bmod \phi(n)$（私钥指数，扩展欧几里得算法）

**加密**：$c = m^e \bmod n$；**解密**：$m = c^d \bmod n$

**为什么安全**：知道 $n$ 但不知道 $p, q$ → 无法计算 $\phi(n)$ → 无法从 $e$ 算出 $d$。分解 2048 位大整数用目前最强算法（数域筛法 GNFS）需要约 $2^{112}$ 次运算——远超人类计算能力。

**量子威胁**：Shor 算法在量子计算机上分解 $n$ 只需多项式时间 $O((\log n)^3)$。RSA 的数学基础是"大数分解困难"——量子计算机会摧毁这个假设。

## 8. 动态规划 → Git diff / DNA 比对

**编辑距离问题**：将字符串 $A$ 转换为字符串 $B$ 所需的最少操作数（插入/删除/替换）。

**递推公式**（Levenshtein 距离）：

$$d[i,j] = \min \begin{cases}
d[i-1,j] + 1 & \text{（删除）} \\
d[i,j-1] + 1 & \text{（插入）} \\
d[i-1,j-1] + (A[i] \neq B[j]) & \text{（替换或匹配）}
\end{cases}$$

**复杂度**：$O(|A| \cdot |B|)$。填满一个 $|A| \times |B|$ 的表格，每个格子 $O(1)$。

**Git diff**：用 Myers 算法（最长公共子序列 LCS 的线性空间变体）计算文件版本的编辑距离 → 输出最小改动集 → 这就是你看到的 `git diff` 输出。

**DNA 比对**：Smith-Waterman 算法在编辑距离基础上加"惩罚缺口"和"奖励匹配"的打分矩阵——把生物学知识编码为数学权重。人类基因组 30 亿碱基对的比对，全靠这套递推公式。
