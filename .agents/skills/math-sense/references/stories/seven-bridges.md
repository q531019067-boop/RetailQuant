# 七桥问题：图论的创世记

**前因**：1736年，普鲁士的柯尼斯堡城有七座桥连接两个岛和两岸。居民间流传一个谜题：能否一次走完所有七座桥，每座桥只走一次，最后回到起点？

**数学方法**：欧拉的天才在于——他不关心桥的长度、位置、形状。他把问题**抽象**为点和线：四块陆地是点，七座桥是线。然后他证明：如果一个点有奇数条线（奇度），这个图不可能存在"欧拉回路"（一笔画）。所有四个点都有奇数度→不可能。**图论**由此诞生。

**游戏启示**：寻路算法（A*）、技能树依赖检查、社交网络分析——都源自欧拉"把现实抽象为点和线"的洞察。

## 数学证明：欧拉路径与欧拉回路

### 问题抽象

将地图抽象为图 $G = (V, E)$：
- 顶点 $V$ = 四块陆地 $\{A, B, C, D\}$
- 边 $E$ = 七座桥
- 每个顶点的**度数** $\deg(v)$ = 连接到该顶点的桥数

柯尼斯堡七桥的度数分布：$\deg(A) = 5, \deg(B) = \deg(C) = \deg(D) = 3$——全部奇数。

### 欧拉定理

**欧拉回路**（起点=终点，每边恰好走一次）存在的充要条件：


$$
\text{每个顶点的度数都是偶数}
$$

**欧拉路径**（起点≠终点，每边恰好走一次）存在的充要条件：


$$
\text{恰好两个顶点的度数为奇数（这两个是起点和终点），其余全为偶数}
$$

### 证明（必要性）

进入一个顶点需要消耗一条边，离开该顶点需要消耗另一条边。因此：
- 对于中间顶点（非起点/非终点）：进入次数 = 离开次数 → 度数为偶数
- 对于起点：离开次数 = 进入次数 + 1 → 度数为奇数
- 对于终点：进入次数 = 离开次数 + 1 → 度数为奇数

若起点=终点（欧拉回路），则每个顶点度数必须为偶数。$\square$

### 应用到柯尼斯堡

全部四个顶点的度数都是奇数 → 不满足任何存在条件 → **不可能一笔画走完七座桥并回到起点**。

欧拉在论文中写道："这个问题与数学无关，它属于位置几何学（geometria situs）"——这是**拓扑学**的诞生宣言。欧拉关心的不是桥的长短弯曲，而是它们如何"连接"——这是抽象思维的典范。
