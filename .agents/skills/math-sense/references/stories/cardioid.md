# 心形线：笛卡尔的数学情书

## 故事

1650年，52岁的法国数学家笛卡尔流亡瑞典，在街头邂逅了18岁的瑞典公主克里斯蒂娜。公主请笛卡尔做她的数学老师。两人在相处中日久生情。

国王发现后大怒，将笛卡尔驱逐。笛卡尔在病中给公主写了最后一封信——信里只有一行公式：


$$
r = a(1 - \cos\theta)
$$

公主取来纸笔，在极坐标纸上逐点描摹。当 $\theta$ 从 $0$ 走到 $2\pi$ 时，一颗完美的心形跃然纸上。

> 这个故事的真实性有争议——笛卡尔确实教过克里斯蒂娜，但教的是哲学。心形线也确实存在。但"情书"这段是后人美好的附会——它已经成了数学浪漫的象征。

## 数学原理：为什么这个方程画出心形

**极坐标方程**：`r = a(1 - cosθ)`，其中 a > 0。

在极坐标系中，r 是点到原点的距离，θ 是角度。关键数学事实：
- 当 θ = 0 时，cos(0)=1，r = a(1-1) = 0 —— 曲线从原点出发
- 当 θ = π/2 时，cos(π/2)=0，r = a —— 到达最右侧
- 当 θ = π 时，cos(π)=-1，r = a(1-(-1)) = 2a —— 到达最左侧，也是曲线最远点
- 当 θ = 3π/2 时，cos(3π/2)=0，r = a —— 回到对称位置
- 当 θ = 2π 时，cos(2π)=1，r = 0 —— 回到原点，闭合

**为什么是心形**：cosθ 在 [0,2π] 上的变化导致 r 从 0→a→2a→a→0，配合θ的旋转，轨迹恰好形成"原点处尖、对面圆"的心形轮廓。尖点出现在 θ=0（r=0），对面最远处在 θ=π（r=2a）。

**直角坐标转换**：x = r·cosθ, y = r·sinθ，代入得：
(x² + y² - ax)² = a²(x² + y²) —— 这是一个四次曲线。

## 心形线的几何性质

**1. 外摆线定义**：心形线是一个圆沿另一个**同样大小**的圆外侧滚动时，圆上一点画出的轨迹。这是它最本质的几何定义。

**2. 弧长**：心形线的完整弧长 = 8a。如果你把a设为1，整条曲线的长度是8——一个非常整洁的数字。

**3. 面积**：心形线围成的面积 = (3/2)πa²。对比：半径为a的圆面积是πa²——心形线的面积恰好是圆的1.5倍。

**4. 光学性质（焦散线）**：如果一束平行光射入圆形杯子的内壁，反射光线的包络线恰好是一条心形线。你在咖啡杯里看到的光斑——底部那个心形的亮纹——就是心形线的焦散现象。你用杯子喝水时，其实每天都在看心形线。

**5. 麦克风指向性**：心形线也是"心形指向麦克风"（Cardioid Microphone）的命名来源——这类麦克风对正前方声音最敏感，对正后方声音几乎不敏感，灵敏度图恰好是心形线方程。

## 动手验证

```
python plot_tools.py --op "polar" --expr "1-cos(theta)" --range "[0,6.28]" --n 200
```

这会画出一条完美的心形线——你会看到θ=0处的尖点、θ=π处的最远点、以及对称的左右两瓣。

> 数学不只是冰冷公式——它可以是情书，可以是你咖啡杯底的光斑，可以是歌手麦克风的方向图。这就是"数感"。
