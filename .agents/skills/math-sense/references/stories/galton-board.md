# 高尔顿板：从"随机舞蹈"中浮现的钟形曲线

## 故事

1873年，英国科学家弗朗西斯·高尔顿（Francis Galton，达尔文的表弟）发明了一个装置：一块倾斜的木板，上面钉着整齐排列的钉子，底部有等宽的竖槽。从顶部漏斗倒入一把小豆子（或铅弹），每颗豆子下落时遇到钉子随机向左或向右弹跳，层层叠叠，最终落入底部的某个槽中。

高尔顿叫它 **Quincunx**（梅花形排列，因钉子按五点梅花状排列），后人称它为"高尔顿板"或"豆子机器"。

围观的人看到豆子在钉阵中弹跳——左、右、左、右——像一群小人在跳舞。当数百颗豆子全部落下后，奇迹出现了：

**底部堆积的豆子，不偏不倚地形成了一条完美的钟形曲线。**

## 数学原理

每颗豆子经过n排钉子，每排独立地选择左或右（概率各50%）。一颗豆子最终落入第k个槽的概率恰好是二项分布：


$$
P(X=k) = \frac{\binom{n}{k}}{2^n}
$$

这正是帕斯卡三角形的第 $n$ 行。当 $n$ 足够大时（高尔顿板通常有 10-20 排），二项分布逼近正态分布——**中心极限定理的完美视觉证明**。

不需要任何数学训练——你用眼睛就能看到"随机→秩序"的魔法。这就是高尔顿的天才。

## 为什么这很重要

高尔顿板不是一个玩具。它是**统计思维**的物理化身：

1. **个体不可预测，群体有规律**——你不知道某颗豆子会落入哪个槽，但你知道所有豆子的分布形状
2. **正态分布不是"自然"的，是"叠加"的——它是大量独立微小随机因素的必然结果
3. **这就是为什么身高、血压、测量误差都近似正态分布——它们都是无数微小因素的叠加**

## 高尔顿的甜豌豆：回归到均值的发现

高尔顿不只发明了高尔顿板。他在研究遗传学时做了另一个著名实验——**甜豌豆实验**。

他收集了不同大小的甜豌豆种子，按大小分组播种。然后测量子代种子的大小。他发现了惊人的规律：

- 大粒种子（直径>平均）的后代 → 倾向于变小，靠近平均
- 小粒种子（直径<平均）的后代 → 倾向于变大，靠近平均

他称这个现象为 **"回归到平庸"**（regression to mediocrity）——后来简称为"回归"（regression）。这是统计学中最重要的概念之一。

**数学上**：子代大小 y 和亲代大小 x 之间的关系是 y = a + bx，其中 0 < b < 1。b<1意味着极端值在下一代中会向均值"回归"。

**游戏启示**：回归到均值是理解和设计"平衡系统"的关键——为什么一个版本OP的武器在下一个版本会被削（回归到正常强度），为什么连赢之后的玩家胜率会下降（回归到真实水平），为什么极端幸运/不幸的事件终将回归平均。

## 为什么这个装置让人过目不忘

高尔顿板中的豆子沿着钉阵弹跳——左、右、左、右——每次弹跳都是随机的。但当上百颗豆子全部落下，底部不偏不倚地堆出一条完美的钟形曲线。这种**直观的视觉冲击**让任何人都能一秒理解中心极限定理——无需数学训练，"随机→秩序"的魔法就在眼前。

这就是高尔顿的天才：用最简单的物理装置演示最深刻的统计规律。
