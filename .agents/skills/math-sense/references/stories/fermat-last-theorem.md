# 费马大定理：350年的执念

## 故事

1637年，法国数学家费马在阅读丢番图的《算术》时，在书页边缘写下一行注记：

> "将一个立方数分成两个立方数之和，或一个四次幂分成两个四次幂之和，或者更一般地将一个高于二次的幂分成两个同次幂之和，这是不可能的。关于此，我发现了一种美妙的证法，可惜这里空白太小，写不下。"

这四行字——**费马大定理**——困扰了人类最聪明的头脑 **350 年**。

## 数学陈述


$$
x^n + y^n = z^n \quad\text{对于 } n > 2 \text{ 没有正整数解}
$$

n=1：显然是 x+y=z，无穷多解
n=2：勾股定理 x²+y²=z²，有无穷多组解（如 3²+4²=5²）
n≥3：费马说没有解——但他没写证明

## 350年的追逐

- 欧拉证明了 n=3 的情况
- 费马本人用"无穷递降法"证明了 n=4
- 索菲·热尔曼（一位女性数学家，不得不以男性笔名发表）证明了一大类n
- 20世纪初，有人悬赏10万马克——收到621封错误证明
- 计算机验证了 n 直到 400万都成立——但这不是证明

**计算机能验证到任何有限数，但不能证明对"所有n"都成立。这就是为什么数学需要证明，而非枚举。**

## 安德鲁·怀尔斯：7年的孤独

1986年，英国数学家安德鲁·怀尔斯得知：**谷山-志村猜想**（关于椭圆曲线和模形式）如果成立，将蕴含费马大定理。怀尔斯从10岁起就梦想证明费马大定理。他做了一个疯狂的决定：

**秘密研究7年，不告诉任何人。**

他担心如果被人知道他在做费马大定理，会有无数人追问进展。他只在已有的椭圆曲线研究框架下发表中间成果作为掩护。他的妻子是唯一知道真相的人。

1993年6月，怀尔斯在剑桥大学的三场讲座最后一刻，写下了证明的最后一行。全场沉默——然后爆发出雷鸣般的掌声。第二天，全世界的报纸头版都是："费马大定理被证明了！"

**但故事还没结束。** 审稿人发现了一个漏洞。怀尔斯又花了一年——和他以前的学生理查德·泰勒一起——修补了漏洞。1995年，最终证明发表。

## 方法论启示

1. **"无穷递降法"**（费马的原创方法）：假设存在最小反例，然后构造更小的反例——矛盾。这是数学归纳法的"反向版"。

2. **桥梁证明**：怀尔斯没有直接攻费马大定理，而是证明了谷山-志村猜想（一个看似不相干的椭圆曲线理论）——然后利用"如果A则B"的逻辑链。**在看似不相关的领域之间建立桥梁，是数学最深刻的方法。**

3. **7年孤独**：怀尔斯的故事告诉我们——有些问题需要"长期沉浸"，而非快速迭代。在一个项目上用足够长的时间深度思考，可能比频繁切换问题更有成效。

4. **计算机验证≠数学证明**：计算机能验证 n=10¹² 都成立，但不能证明"对所有n成立"。这区分了"经验证据"和"数学真理"。

## 与游戏的连接

- **数值验证的局限**：掉率模拟10000次看起来正确≠公式正确。需要解析证明（或用贝叶斯方法量化不确定性）
- **等价变换**：如果直接求解太难，找等价问题——就像怀尔斯把费马大定理转化为椭圆曲线问题
- **独立验证**：即使99.999%确定，也需要第三方独立验证（就像审稿人发现了怀尔斯的漏洞）
